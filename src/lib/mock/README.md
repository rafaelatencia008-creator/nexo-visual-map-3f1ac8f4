# src/lib/mock

Dados fictícios, personas, fixtures e serviços simulados usados pela fase visual do projeto.

Serão criados a partir do bloco **LV-07**. Todo conteúdo deve ser previsível, fictício, sem PII e substituível por chamadas reais no futuro (após LV-18).

**Não colocar aqui:** dados reais, segredos, chaves, tokens, credenciais, clientes Supabase, chamadas `fetch` externas.
