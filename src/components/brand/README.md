# Marca — Nexo Pericial 360

Componentes de identidade visual (logotipo, símbolo). Não contêm lógica de negócio.

## Logo

- `<Logo />` — símbolo + wordmark completo (para cabeçalhos, telas de login).
- `<Logo variant="mark" />` — apenas o símbolo (para favicon, avatares, espaços reduzidos).
- Aceita `className` para dimensionamento (ex.: `className="h-10"`).
- Usa tokens de cor (`--primary`, `--brand-accent`, `--foreground`) e a fonte `--font-display` (Fraunces).
