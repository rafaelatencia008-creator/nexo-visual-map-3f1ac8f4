# src/components/ui

Componentes base do design system (botões, campos, cards, feedback, diálogos, navegação, layout).

Serão criados no bloco **LV-02**. Devem ser genéricos, reutilizáveis e usar somente tokens semânticos definidos em `src/styles.css`.

**Não colocar aqui:** telas, componentes de marketing, componentes específicos do `/app`, dados fictícios, chamadas de rede.
